Documentación API GreenPoint

El API de GreenPoint te permite obtener en tu web o aplicacion los datos de nuestra plataforma. En este apartado encontraras toda la información necesaria para obtener los datos necesarios dentro de la aplicacion que vayas a desarrollar.

Descripción

Los datos que puedes manejar usando nuestro API son:

  • Usuarios: Cada usuario tiene su propia clave para poder realizar las consultas necesarios dentro de la plataforma.
  • Favoritos: Lista de favoritos de cada usuario, requiere la clave de usuario para la obtencion de los datos.
  • Comentarios: Comentarios de los usuarios a cara contenedor, en este caso solo es neceario la clave del usuario para registrar nuevos comentarios.

Ademas los datos pueden ser obtenidos en dos formatos diferentes: JSON y XML.

Para ello solo tendremos que añadir el atributo en la url de la peticion de la siguiente manera:

    Formato JSON - URL/ApiGreenpoint/(dato-a-consultar)?formato=json

    Formato XML - URL/ApiGreenpoint/(dato-a-consultar)?formato=xml

Usuarios

Al realizar una consulta sobre los usuarios el El tipo Usuario tiene como atributos los siguientes campos:

  • nombre: Nombre del usuario.
  • contrasena: Contraseña encriptada.
  • email: Correo del usuario.
  • claveApi: Clave unica de acceso para el usuario.

La siguiente tabla muestra como registrar y loguear usuarios:

 Metodo  URL  Parametros  Retorno  Descripcion
 POST  URL/ApiGreenpoint/v1/usuarios/registro  JSON{nombre,correo,contrasena}  nombre, correo, claveApi  Registra un nuevo usuario
 POST  URL/ApiGreenpoint/v1/usuarios/login  JSON{correo,contrasena}  nombre,correo,claveApi  Loguea un usuario

Favoritos

El tipo Favorito tiene como atributos los siguientes campos:

  • idFavorito: Identificador unico para cada favorito.
  • idContenedor: Identificador unico del contenedor.
  • tipo: tipo del contenedor.(ver tipos)

La siguiente tabla muestra como obtener y crear favoritos:

NOTA: se necesita la claveApi del usuario para obtener favoritos.
NOTA 2: Añadir la clave en el Header con tipo Authorization

 Metodo  URL  Parametros  Retorno  Descripcion
 GET  URL/ApiGreenpoint/v1/favoritos  -   favoritos[]  Obtener todos los favoritos del usuario
 GET  URL/ApiGreenpoint/v1/favoritos  ?tipo=(tipo)  favoritos[]  Obtener los favoritos del usuario de un tipo
 POST  URL/ApiGreenpoint/v1/favoritos  JSON{idContenedor,tipo}  idFavorito  Crea un nuevo favorito

Comentarios

El tipo Comentario tiene como atributos los siguientes campos:

  • nomAutor: Nombre del usuario que ha realizado el comentario.
  • idContenedor: Identificador unico del contenedor.
  • tipo: Tipo del contenedor.(ver tipos)
  • fecha:Fecha de publicacion
  • titulo: Titulo del comentario
  • texto: Cuerpo del comentario

La siguiente tabla muestra como obtener y crear comentarios:

NOTA: se necesita la claveApi del usuario para crear comentarios.
NOTA 2: Añadir la clave en el Header con tipo Authorization

 Metodo  URL  Parametros  Retorno  Descripcion
 GET  URL/ApiGreenpoint/v1/comentarios  ?contenedor=(id)&tipo=(tipo)  comentarios[]  Obtener todos los comentarios de un contenedor
 POST  URL/ApiGreenpoint/v1/comentarios  JSON{idContenedor,tipo,fecha,titulo,texto}  idComentario  Crea un nuevo comentario
